## [Voting System](voting_system_recreation.py)
This is a recreation of my school's voting system. It is not an exact copy, but rather a similar project. I limited myself to using only things taught in school: `if/elif/else`, `for`/`while` loops, `print()`, lists, strings, `int`, `float`, arithmetic operators, relational operators, variables, augmented assignment operators, and typecasting. 

Now, if you are wondering why I used the `len()` function and `.append()` list method, it is because I allowed myself an exception of any three external functions/methods outside of my classroom restrictions.
